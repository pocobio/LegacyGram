# Chaquopy Java Type Stubs

This directory contains Python Interface (`.pyi`) files for the **Chaquopy v16** `java` module.
For reference, see the official [Python API documentation](https://chaquo.com/chaquopy/doc/16.0/python.html).

### Context
I couldn't find any existing type definitions for Chaquopy v16, 
so I created them specifically to simplify the development of ExteraGram plugins.

### Purpose
These files provide **IntelliSense**, **autocompletion**, and **type checking** for Java primitives (like `jint`, `jfloat`, `jboolean`)
and utilities (`jclass`, `cast`) within your IDE during development.
