# LegacyGram Plugin
A highly customizable plugin for [exteraGram](https://exteragram.app) to remove paid and bloated features from the Android Telegram client

## Building
1. Clone the repo:
```bash
git clone https://github.com/repinek/LegacyGram.git
cd LegacyGram
```
2. Install dependencies using [uv](https://docs.astral.sh/uv/):
```bash
uv sync
```
3. Build
```bash
uv run build.py
```
Result of building will be saved at `dist/LegacyGram.plugin`

## Installation
1. Send `LegacyGram.plugin` to any Telegram chat (e.g., **Saved Messages**).
2. Tap on the file in the chat and tap **Install**.
